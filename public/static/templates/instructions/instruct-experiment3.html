Instructions



You don't need to learn where goals are on the map. A goal's letter (either A, B, C or D) determines its value and the location is not important.

You will see the same colors repeated many times, so try to learn the goals as best you can! This will make the game much easier.

Keep in mind that if you are able to learn which goals go with which colors, you will recieve a much bigger bonus than if you don't.

When you finish the game there will be a short questionnaire. Remember! use the a, s, d, f, j, k, l, ; keys to move around the maze.
